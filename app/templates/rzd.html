Monogram Bookmark, Sticker, and Coaster Set

Another project for a semester-long class on color and communication was the following set of a bookmark, sticker, and coaster. I first used my initials to create a monogram, focusing on positive and negative space, transparency, and color interaction. Then, I moved onto focusing on the actual "canvas" of the object (a long vertical strip for the bookmark, a sqaure for the sticker, etc.) and how the background that both the monogram and the object itself were on affected the way the color looked.

Mockups for each of these objects were laser color printed and shown in context through photography. To the left are those context shots, as well as the final set, many in progress versions, and some examples of early monogram designs and palettes.
